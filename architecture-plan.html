04 — Razorpay Payment Flow (Step by Step)
1. User fills checkout form + clicks Pay
Name, email, phone validated on the client. Button disables to prevent double-clicks.
2. Razorpay popup opens
User pays via UPI / Card / NetBanking / Wallet. Red theme matches your brand. Prefilled with their name, email, phone.
3. Razorpay confirms payment (client-side handler)
Returns razorpay_payment_id, razorpay_order_id, razorpay_signature to the browser.
4. Redirect to thank-you page
Browser goes to thank-you.html?payment_id=pay_ABC&name=Rahul&email=rahul@gmail.com. Thank-you page reads these and shows personalized confirmation.
5. Razorpay sends Webhook to your server Server-side
Razorpay POSTs a payment.captured event to your backend. Your server verifies HMAC signature, creates user in Supabase, sends welcome email via Resend.
6. Student gets welcome email + can access dashboard
Email from Resend with login link. Student clicks Go to Dashboard → watches modules.
Note on phases: The current 4 HTML files handle steps 1–4 completely (everything in the browser). Steps 5–6 require a backend (Next.js + Supabase + Resend). For a quick launch, you can skip the backend — students just use the dashboard directly after the thank-you page, and you manually email login info.

07 — Deployment Checklist
🚀 Quick Deploy (Netlify Drop — 5 minutes)
  • Create a folder called youtube-blueprint
  • Put all 4 HTML files in it: sales-page.html, checkout.html, thank-you.html, dashboard.html
  • Go to netlify.com/drop
  • Drag the folder into the browser
  • Site goes live at random-name.netlify.app
  • Add custom domain in Netlify settings
⚙️ Before Going Live — Required Steps
  • Open checkout.html → replace rzp_test_XXXXXXXX with your live Razorpay key
  • Open thank-you.html → replace https://t.me/YOUR_TELEGRAM_GROUP with real Telegram link
  • Open dashboard.html → replace all vimeoId fields with real Vimeo video IDs
  • Upload course videos to Vimeo and set domain whitelist to your live domain
  • Test the full flow: Sales → Checkout → Pay (test mode) → Thank You → Dashboard
  • Switch Razorpay from test key to live key
✅ Optional (but recommended)
  • Add Google Analytics or Hotjar to track visitor behaviour
  • Add Facebook Pixel for retargeting ads
  • Set up Razorpay webhook → Supabase → Resend for automated welcome emails
  • Add OTP login via Supabase Auth for returning students
  • Add a proper index.html that redirects to sales-page.html
